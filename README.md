# 🧠 Prajñāvan – AI Powered Virtual Assistant

An AI-powered virtual assistant chatbot with voice interaction, NLP, and system automation capabilities.

---

## 🚀 Overview

**Prajñāvan** is an intelligent conversational assistant designed to provide human-like, context-aware interactions.  
It integrates **Natural Language Processing (NLP)**, **voice recognition**, and **system automation** to create a powerful and interactive AI assistant.

The project is built using a **full-stack architecture** combining Python backend with a modern web-based frontend.

---

## ✨ Features

- 🗣️ Voice Recognition (Speech-to-Text)
- 🔊 Text-to-Speech Response System
- 💬 Intelligent Chatbot with NLP
- ⚙️ System Automation (open apps, commands, etc.)
- 🌐 Web-based Interactive UI
- 🎨 Real-time animations and visual feedback
- 🧠 Context-aware AI responses
- 🌍 Multilingual support (basic)

---

## 🏗️ Tech Stack

### 🔹 Backend
- Python
- Flask
- HugChat API (AI responses)
- SQLite (Database)

### 🔹 Frontend
- HTML5
- CSS3
- JavaScript

### 🔹 Other Technologies
- Speech Recognition (Google API)
- Text-to-Speech (pyttsx3)
- Eel (Python ↔ JS communication)

---

## 🧩 System Architecture

The system follows a **3-tier architecture**:

- **Frontend (View)** → User Interface (HTML, CSS, JS)
- **Backend (Controller)** → Flask + Eel communication
- **Core Logic (Model)** → NLP, AI processing, automation

## ⚙️ Installation & Setup

### 🔹 1. Clone Repository
```bash
git clone https://github.com/your-username/prajnavan-ai-assistant.git
cd prajnavan-ai-assistant
```

### 🔹 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 🔹 3. Run the Application

```bash
python main.py
```
